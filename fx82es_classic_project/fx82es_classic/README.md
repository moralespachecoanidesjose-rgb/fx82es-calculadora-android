FX-82ES Classic - APK listo para compilar
=========================================
Este proyecto recrea la calculadora CASIO fx-82ES en una app Android minimalista
(sin historial ni funciones extra). Está listo para abrir en Android Studio y
generar un APK compatible con Redmi Note 12.

Instrucciones:
1. Descargar y descomprimir.
2. Abrir en Android Studio.
3. Build > Build APK(s).
4. Instalar en el teléfono (activar 'Instalar apps desconocidas' si es necesario).

Nota: La app es 100% Kotlin/Android (sin librerías nativas), por lo que es compatible
con ARM64 (Redmi Note 12) cuando compiles en Android Studio.
